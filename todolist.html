Upon opening the website, we can chose 4 pre-defined catagory, and to go to list of our choice, click on go to list button.

The connection between our list and list content is established via React Router Dom.

Once select a list, user will be greeted with a textbox, where the user can enter a new task on the top.

Add task was achieved using useState and onClick method to put new task into a task Array, in which we display individual task by javascript mapping to a JSX element

Also used localstorage of the browser to store the task we input

For example, the user input "make diner" in the input box. Upon hitting enter, make dinner would show up in our task list.

And I already added task "sweep the floor" to demonstrate the ability to display task instead of a placeholder if there is new task.

The app also do not allow user to input duplicated task on the input bar and also prevent empty input.

To edit a task, first hit the edit button next to the text. The form to edit task will then show up above the task.

Edit the task by typing in the input, and click edit task to edit.

The edit input prevent empty input.
