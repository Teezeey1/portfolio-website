First run the server with npm start command, then open the app and we are greeted with the sign in page

Enter any user name with at least 3 characters to proceed

The server was running socket.io and it handles user connecting,disconnecting, and emitting events

By simply typing in the text box, we can send message via the send button

onClick method was used to handle click event, where we send an event to the server using the socket.

Lastly, but hitting the dark theme button, we can enable dark theme

This was achieved by useState method where we toggle on and off light theme
